# 简介

欢迎来到 **UniRTM (通用运行时管理器)** 官方文档。

UniRTM 受到 `mise` 和 `asdf` 等工具的启发，是一款企业级的底层工具集及运行时管理器。它能够将你的开发环境、任务流以及环境变量统一由一个极其快速的 Go 语言可执行文件来管理。

## 为什么选择 UniRTM？

在现代软件开发中，每个项目往往需要多种运行时（如 Node.js, Go, Python）、各种 CLI 工具（如代码检查工具、格式化工具）以及特定项目的环境变量。传统情况下，开发者需要混合使用 `nvm`、`pyenv`、`direnv` 和 `make` 等各种工具来管理它们。

UniRTM 完美替代了这一切：

- **多语言工具管理器**：自动安装并管理多种编程语言及配套工具的版本。
- **环境管理**：在进入项目目录时，自动加载所需的环境变量。
- **任务运行器**：内置简单易用的跨平台任务执行器。
- **极致快速**：采用 Go 语言编写，提供最大化性能。

## 快速开始

要开始使用 UniRTM，请按照代码仓库 README 中的安装指南进行安装。如果您使用的是 macOS 或 Linux，您可以直接运行通用安装脚本：

```bash
# 示例安装脚本
curl -sL https://raw.githubusercontent.com/snowdreamtech/UniRTM/main/install.sh | bash
```

安装完成后，您就可以使用 `unirtm` 命令来安装开发工具、运行日常任务和管理环境了。
